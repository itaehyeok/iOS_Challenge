## 챌린지: AutoLayout

**ViewController.swift에 사각형 모양의 서브뷰를 만들어서 오토레이아웃 주기**

예시 이미지

### 조건
새로운 서브뷰의 색상을 주어 슈퍼뷰와 구분하기(색상을 상관없음)
서브뷰는 ViewController의 가운데에 위치(가로, 세로 전부)
서브뷰의 크기는 가로 100, 세로 100
